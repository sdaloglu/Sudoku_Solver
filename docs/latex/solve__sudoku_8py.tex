\doxysubsection{Detailed Description}
Main .py file for sudoku solving backtracking algorithm.

Backtracking algorithm is a brute force method which tries all possible values for the empty cells until it finds the correct one. The algorithm reverts back to the previous configuration of the sudoku table as soon as a contradiction is found with the current configuration. \begin{DoxyAuthor}{Author}
